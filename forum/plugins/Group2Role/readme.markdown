# Group2Role
Group2Role plugin for vanillaforums. Give a role to every user in a group.

Requirements 
* [Memberships](http://vanillaforums.org/addon/memberships-plugin)
* [Groups](http://vanillaforums.org/addon/groups-plugin) plugins
* Vanilla > 2.0.18
